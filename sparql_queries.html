  5. Label Transition Impact: What are the differences in themes and topics before and after Taylor Swift transitioned from Big Machine label to Republic and her own production (Taylor Swift Production)?

    6. 
                        q6="""
                        SELECT ?label ?topic
                        WHERE {
                        { SELECT ?label ?topic
                        WHERE {
                        ?album rdf:type :Album .
                        ?album :belongsToLabel ?label .
                        ?album :hasTopic ?topic .
                        FILTER (?label = :Big_Machine)
                        }
                        }
                         UNION
                        { SELECT ?label ?topic
                        WHERE {
                        ?album rdf:type :Album .
                        ?album :belongsToLabel ?label .
                        ?album :hasTopic ?topic .
                        FILTER (?label = :Republic)
                        }
                        }
                        UNION
                        { SELECT ?label ?topic
                        WHERE {
                        ?album rdf:type :Album .
                        ?album :belongsToLabel ?label .
                        ?album :hasTopic ?topic .
                        FILTER (?label = :Taylor_Swift_Production)
                        }
                        }
                        }
                        """
    Results:
    - For the label "Big Machine": the albums from this label cover a wide range of topics including desire, enjoying life, moving on, relationships, romantic vibes, turmoil, appreciation, breakup, expression, intimacy, music, nostalgia, conflict, decision-making, future, love, regret, uncertainty, challenges, loneliness, being emotional, enthusiasm, fantasy, persistence, reflection, adventure, nature, return and departure.
    - For the label "Republic": the albums from this label also cover diverse topics such as being emotional, frustration, nostalgia, reflection, struggle, suffering, desperation, discovery, pain, anger, desire, devotion, excitement, relationships, aspirations, companionship, family, memories, seeking understanding, heartache, intensity, reconciliation, and reminiscence.
    - For the label “Taylor’s Swift Productions”: frustration, nostalgia, reflection, struggle, suffering, desperation, discovery, pain, anger, desire, devotion, excitement, relationships, aspirations, companionship, family, memories, seeking understanding.
