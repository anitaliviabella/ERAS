Workflow

The workflow of the project is divided mainly in four steps:

1. Formulation of research and competency questions:

As already explained, the general purpose is for the project to reflect all the 11 eras in terms of topics to then satisfy some curiosities about their evolution. The query part will be accomplished in the last step of the project which will give objective conclusions to more precise questions.

2. Knowledge extraction:

The second step has been the extraction of the main topics though topic modeling. In the first place, comprehensive datasets have been gathered from this source , which allowed to work on well-organized data, containing all the lyrics for each album.

3. Ontology development:

The third step involves the development of the ontology using Protégé.

4. SPARQL queries:>

The last sep concerns the exploitation of the ontology through SPARQL queries. THe competency questions in natural language have been converted in SPARQL syntax in order to query the ontology and obtain answers.
