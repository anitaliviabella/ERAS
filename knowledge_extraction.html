Topic modeling


The following python code is designed to analyze the collection of song lyrics of the album folklore and categorize them into thematic topics using a technique called Non-negative Matrix Factorization (NMF).
The process begins by reading the lyrics from CSV files and preprocessing them to remove irrelevant words. Stopwords, common words like "the" or "and," are eliminated to focus on meaningful content.
Next, the lyrics are converted into a numerical representation using TF-IDF (Term Frequency-Inverse Document Frequency) vectorization. This step assigns weights to words based on their importance in individual songs and across the entire corpus. The NMF algorithm is then applied to decompose the TF-IDF matrix into a set of topics. Each topic represents a collection of words that commonly occur together in the lyrics, capturing underlying themes within the music.
Each song is categorized into one or more topics based on its thematic content, and the results are saved to a CSV file for further analysis. The same procedure is applied for all the 11 albums in order to obtain the topics for each song and, consequently, for each album.

The reasons behind the choice of this technique rely on its intuitiveness: by decomposing the IDF matrix into non-negative factors, NMF ensures that each topic and its associated words have non-negative weights, making the results more interpretable. Additionally, NMF works well with sparse and high-dimensional text data, which is consistent with the features of song lyrics. Moreover, NMF allows for flexibility in the number of topics to be extracted, which is ideal for making reasonable choices for the chosen dataset, which is quite large considering it contains all the lyrics of the artist's music production. Applying NMF consistently across multiple albums ensures a standardized approach to thematic analysis, allowing for meaningful comparisons. In fact, using the same technique across all the albums allows us to compare thematic evolution, diversity, and commonalities. In addition to its technical advantages, in my personal opinion and interpretation of the artist's lyrics, NMF provided meaningful results: the topics generated by NMF aligned with the themes I perceived in the lyrics.
However, this might create some disagreement due to subjective interpretation, but the interest lies in understanding how the machine aligns with our perception.

Text Vectorization: The TF-IDF (Term Frequency-Inverse Document Frequency) vectorizer from sklearn is applied to convert the lyrics into a numerical representation. This process removes stopwords and ignores words appearing in less than 10% of the documents (songs).

tfidf = vectorizer.fit_transform(songs['lyric'])

Topic Modeling with NMF: Non-negative Matrix Factorization (NMF), a dimensionality reduction technique, is utilized to extract latent topics from the TF-IDF matrix. The model is set to identify six topics within the lyrics.

In [ ]:
from sklearn.decomposition import NMF
In [ ]:
nmf = NMF(n_components = 6)
In [ ]:
topic_values = nmf.fit_transform(tfidf)

Topic Interpretation: The script prints the top 10 most relevant words associated with each of the six identified topics, aiding in the interpretation of the thematic content.

In [ ]:
for topic_num, topic in enumerate(nmf.components_):
message = "Topic #{}: ".format(topic_num + 1)
message += " ".join([vectorizer.get_feature_names_out()[i] for i in topic.argsort()[:-11 :-1]])
print(message)
